# ATM Management System (Java)

## Description
A console-based ATM application developed using Core Java and OOP concepts.
The system authenticates users and allows deposit, withdrawal, transfer,
and transaction history viewing.

## Technologies Used
- Java
- OOP Concepts
- Collections

## Features
- User authentication
- Transaction history
- Deposit & withdrawal
- Fund transfer
